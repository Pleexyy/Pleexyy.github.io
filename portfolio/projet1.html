Projet C (décembre 2018)

Jeu graphique en C

A PROPOS DU PROJET

Ce projet a été réalisé dans le cadre de ma première année d'IUT Informatique.

Mon projet consiste à créer un jeu nommé Memory en langage C, en utilisant l’interface graphique de l’IUT. Au début de la partie, toutes les cartes sont face cachée. A chaque tour, le joueur découvre 2 cartes, doit bien visualiser celles-ci, tenter de mémoriser leur emplacement, puis les cache à nouveau. Le but étant de retourner successivement les 2 cartes identiques formant la paire afin de les remporter. La partie est terminée quand toutes les paires ont été trouvées. Pour retourner les cartes il suffit d’un simple clic de souris sur celles-ci.

Fonctionnalités du programme.

Le programme se décompose en trois sections : le menu de sélection de niveau, le plateau de jeu avec le nombre de tours joués, et l’écran de fin lorsque la partie est terminée.

Le menu principal.

Lors du lancement du programme, on tombe sur le menu, où trois choix de niveaux de difficultés nous sont proposé :

- Débutant

- Intermédiaire

- Expert

Le terrain de jeu

Après avoir choisi le niveau de difficulté, nous sommes redirigé sur une grille qui correspond au niveau choisi :

- Grille 2 par 2 pour ​ Débutant

-Grille 4 par 4 pour ​ Intermédiaire

- Grille 6 par 6 pour ​ Expert

La partie commence, le chronomètre est lancé et les cartes sont placées. En haut à droite est inscrit le nombre de tours joués par l’utilisateur.

Pour finir, voici la page de victoire qui affiche un message disant que le jeu est terminé; ainsi que le temps écoulé à la résolution du jeu.
